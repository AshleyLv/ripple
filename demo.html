Ripple

Hey, this is a jQuery plugin, which is used to decorate element with ripple effect. ^O^

Html

<button class="rippleBtn">click me</button>

Javascript

$('.rippleBtn').ripple({
  color:'rgba(45, 161, 196, .3)' //custom ripple color
  duration: 750, //custom ripple duration
  timingFunction:'linear' //custom ripple timing function
});
